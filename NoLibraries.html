Missing libraries


Welcome to DraiWiki! Thank you for using our software. Before you can start using your freshly-installed wiki, there are a few more things you need to do. We've detected that several important libraries are missing. No worries, that's normal. It is, however, necessary that you install them, as DraiWiki won't work without them.

To make things easier, we've created a quick guide:

  1. Install the Composer and NPM package managers.
  2. Go to the main directory. This is the directory that contains the src and public folders, as well as several others.
  3. Enter a terminal or command prompt, and execute the following command composer install
  4. Execute the npm install command
  5. Verify that both the vendor and node_modules directories exist.
  6. Refresh this page and enjoy your DraiWiki installation

Are you receiving this message in error? If so, we apologize for the inconvenience. For some reason, DraiWiki did not detect the vendor and node_modules directories (or you're accessing this error file directly). Make sure you have ran Composer and NPM in the directory that contains the admin, lang, public and src directories, and that all files are accessible.

Good luck with your wiki installation! Make us proud.
